# Personal Portfolio

Welcome to my fully responsive portfolio website that displays my programming projects and skills, as well as some small trinkets to get to know me better, like the song I'm currently listening to on Spotify, or pictures of me and my friends.

## Tech Stack
Front-End: 
- React, JavaScript, Tailwind CSS, Daisy UI, Motion

Deployment: 
- Vercel

API:
- Spotify Web Playback SDK
